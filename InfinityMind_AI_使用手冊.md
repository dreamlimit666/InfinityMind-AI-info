## 系統簡介

**InfinityMind AI** 是一個企業級的 AI 知識問答系統，支援：
- 🤖 多機器人實例（5 個獨立 Bot）
- 💬 多渠道整合（LINE / Teams / WebChat）
- 📚 RAG 知識檢索
- 🔐 完整權限管理（用戶/角色/資料夾/文檔分級）
- 🖼️ 圖片辨識功能（Azure Vision API）
- 🌓 深色/淺色主題切換- 🌐 **內外部共用**：同時支援內部員工與外部訪客，灵活切換驗證模式

### 3.2 新增使用者 - 權限設置

![新增使用者-權限設置](../image/新增使用者-權限設置.png)

**權限配置項目：**

#### 📁 資料夾權限
- 多選資料夾列表
- 決定使用者可存取哪些知識庫
- 支援細粒度控制

#### 🤖 Bot 訪問權限
- 勾選可使用的 Bot 實例
- 支援 Bot 1 ~ Bot 5 獨立授權

#### 🛡️ 角色指派
- 管理員（Admin）
- 一般用戶（User）
- 訪客（Guest）

**權限繼承規則：**
```
管理員 > 一般用戶 > 訪客
   ↓         ↓        ↓
 全部權限  指定權限  唯讀
```

---

### 3.3 新增使用者 - 安全許可

![新增使用者-安全許可](../image/新增使用者-安全許可.png)

**安全分級制度（4 級）：**

| 級別 | 名稱 | 標籤 | 說明 |
|------|------|------|------|
| **Level 1** | 公開 | 🌐 Public | 無限制文檔 |
| **Level 2** | 內部 | 🏢 Internal | 企業內部文檔 |
| **Level 3** | 機密 | 🔒 Confidential | 敏感業務資料 |
| **Level 4** | 極機密 | 🔐 Secret | 最高機密文檔 |

**設定項目：**
- 📋 **安全許可級別**：選擇 1-4
- 📅 **授權日期**：許可生效時間
- ⏰ **過期日期**：許可失效時間（可選）
- 👤 **授權人員**：記錄核准者
- 📝 **授權原因**：稽核用途

**存取規則：**
```
使用者許可級別 ≥ 文檔分級 → 可存取
使用者許可級別 < 文檔分級 → 拒絕存取
```

## 機器人設定

### 4.1 多組 Bot 配置

![機器人設定-可設定多組bot](../image/機器人設定-可設定多組bot.png)

**支援多 Bot 實例：**
- 🤖 **Bot 1 ~ Bot 5**：5 個獨立機器人
- 🎯 **獨立配置**：各自擁有專屬設定
- 🔑 **獨立認證**：每個 Bot 有獨立的 Azure Bot Service 憑證

**適用場景：**
- 📊 **部門隔離**：不同部門使用不同 Bot
- 🌍 **多語言支援**：中文 Bot / 英文 Bot
- 🎭 **角色專業化**：HR Bot / IT Bot / Sales Bot

---

### 4.2 資料夾掛載與驗證需求

![機器人設定-可設定掛載資夾與使用者驗證需求](../image/機器人設定-可設定掛載資夾與使用者驗證需求.png)

**配置項目：**

#### 📁 掛載資料夾
- 多選資料夾列表
- 決定此 Bot 可檢索哪些知識庫
- 支援動態調整

**範例配置：**
```
Bot 1 (HR)    → 掛載：人事資料夾、公告資料夾
Bot 2 (IT)    → 掛載：技術文檔、操作手冊
Bot 3 (Sales) → 掛載：產品資訊、銷售資料
```

#### 🔐 使用者驗證需求
- **啟用驗證**：要求使用者登入後才能對話
- **停用驗證**：允許訪客模式（匿名對話）

**決策建議：**
```
內部敏感資料 Bot → 啟用驗證 ✅
公開客服 Bot     → 停用驗證 ❌
```

## 文檔與資料夾管理

### 5.1 資料夾管理 - 啟用狀態與公開性

![資料夾管理-設定啟用或關閉, 資料公開或內部](../image/資料夾管理-設定啟用或關閉%2C%20資料公開或內部.png)

**資料夾屬性配置：**

#### 🔄 啟用/停用
- **啟用**：Bot 可檢索此資料夾的文檔
- **停用**：暫時隱藏，不影響文檔本身

#### 🌐 公開/內部
- **公開（Public）**：適用於 FAQ、使用說明等
- **內部（Internal）**：限內部員工存取

#### 🎨 資料夾屬性
- 📝 **名稱**：資料夾顯示名稱
- 🏷️ **描述**：用途說明
- 📊 **預設分級**：新增文檔的預設安全級別
- 📁 **文檔數量**：即時統計

### 5.2 資料夾內文件管理

![資夾內文件管理](../image/資夾內文件管理.png)

**文件列表功能：**
- 📄 **檔案名稱**：上傳的原始檔名
- 📅 **上傳日期**：建立時間
- 👤 **上傳者**：記錄操作人員
- 📏 **檔案大小**：空間使用統計
- ⚙️ **操作**：編輯、刪除、下載

**支援檔案格式：**
```
文件：.pdf, .docx, .txt, .md
試算表：.xlsx, .csv
簡報：.pptx
```

---

### 5.3 文件分級管理

![資夾內文件管理-文件分級](../image/資夾內文件管理-文件分級.png)

**單一文件安全分級：**

每個文檔可設定獨立的安全級別：
- 🌐 **Level 1 - Public**：公開文檔
- 🏢 **Level 2 - Internal**：內部文檔
- 🔒 **Level 3 - Confidential**：機密文檔
- 🔐 **Level 4 - Secret**：極機密文檔

**分級變更記錄：**
- 📝 **變更時間**：自動記錄
- 👤 **操作人員**：稽核追蹤
- 💬 **變更原因**：必填說明

**分級影響：**
```
使用者查詢時，系統會：
1. 檢查使用者安全許可級別
2. 過濾出可存取的文檔
3. 僅檢索符合權限的內容
```
